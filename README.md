# Java AWS Deployment Demo

[![Java CI with Maven](https://github.com/Sowe-coder/java-aws-deployment-demo/actions/workflows/deploy.yml/badge.svg)](https://github.com/Sowe-coder/java-aws-deployment-demo/actions/workflows/deploy.yml)

A demonstration project showcasing deployment of a Java Spring Boot application on AWS infrastructure, including EC2/Elastic Beanstalk, RDS, and IAM security best practices.

## 🚀 Architecture Overview

- **Application**: Spring Boot REST API
- **Compute**: AWS EC2 or Elastic Beanstalk
- **Database**: AWS RDS (MySQL/PostgreSQL)
- **Security**: IAM Roles and Policies
- **Deployment**: Manual scripts and CI/CD pipeline

## 📋 Prerequisites

- Java 11 or higher
- Maven 3.6+
- AWS Account (free tier eligible)
- AWS CLI configured
- MySQL/PostgreSQL client (optional)
